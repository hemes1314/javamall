create 创建订单API

  • 输入参数
  • 返回结果
  • 返回示例
  • API工具

创建订单，购物车必须有数据，如果当前用户登录则会将此订单记录为此会员的订单，如果没登录则会生成匿名订单

API地址：

/api/shop/order!create.do?

输入参数

名称 类型 是否必须 示例值 默认值 描述
address_id int 必须 收货地址ID
payment_id int 必须 支付方式ID
shipping_id int 必须 配送方式ID
shipDay String 可选项 配送时间
remark String 可选项 备注

返回结果

名称 类型 是否必须 默认值 描述
result int 为1表示添加成功0表示失败
