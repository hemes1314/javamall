会员操作API

提供了会员登录、会员注销登录、修改会员密码、验证原密码是否正确、搜索会员的功能

API列表

  • login
    会员登录
  • logout
    会员注销登录
  • changePassword
    修改会员密码
  • password
    验证原密码是否正确
  • search
    搜索会员
  • saveinfo
    保存会员资料

数据结构列表

  • member
    会员
