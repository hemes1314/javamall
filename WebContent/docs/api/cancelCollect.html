删除已经收藏的一件商品（要求已经登录会员）

API地址：

/api/shop/collect!cancelCollect.do?favorite_id=1

输入参数

名称 类型 是否必须 示例值 默认值 描述
favorite_id int 必须 要删除收藏的商品的ID

返回结果

名称 类型 是否必须 默认值 描述
result Favorite 必须 如果等于“0”，删除错误，如果等于“1”，那么表示删除成功
