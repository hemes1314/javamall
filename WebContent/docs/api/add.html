评论和咨询

API地址：

/api/shop/commentApi!add.do?goods_id=1&commenttype=1&content=测试

输入参数

名称 类型 是否必须 示例值 默认值 描述
goods_id int 必须 商品id
commenttype int 必须 1,2 评论类型,可选值：1或2，1为评论，2为咨询
content String 必须 评论内容
file File 选填 评论的图片

返回结果

名称 类型 是否必须 默认值 描述
result int 必须 1 如果等于“0”，评论错误，如果等于“1”，那么表示评论成功
message String 返回信息
