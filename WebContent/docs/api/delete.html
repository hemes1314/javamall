删除购物车的一项（或者说是删除购物车的某个商品）

API地址：

/api/shop/cart!delete.do?cartid=1

输入参数

名称 类型 是否必须 示例值 默认值 描述
cartid int 必须 删除购物车里商品所对应的ID

返回结果

名称 类型 是否必须 默认值 描述
result CartItem 必须 1 如果等于“0”，删除错误，如果等于“1”，那么表示删除成功
