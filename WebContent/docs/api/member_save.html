保存会员资料

API地址：

/api/shop/member!saveinfo.do

输入参数

名称 类型 是否必须 示例值 默认值 描述
name String 必须 真实姓名
faceFile File 会员头像
sex String 会员性别
mybirthday String 会员出生年月
address String 会员联系地址
zip String 邮编
mobile String 手机
tel String 固定电话

返回结果

名称 类型 是否必须 默认值 描述
result int 必须 1 如果等于“0”，编辑资失败，如果等于“1”，编辑资料成功
message String 返回信息
