获取购物车数据

API地址：

/api/shop/cart!getCartData.do?

输入参数

名称 类型 是否必须 示例值 默认值 描述
无

返回结果

名称 类型 是否必须 默认值 描述
result String 必须 如果等于“0”，表示调用失败，如果等于“1”，那么表示调用成功，并返回total值和count值
data.total String 如果成功，返回total等于购物车总价
data.count String 如果成功，返回count值，即购物车内商品总数
