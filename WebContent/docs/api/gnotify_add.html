商品的缺货登记。此API必须在会员登录后才可使用。

API地址：

/api/shop/gnotify!add.do?goodsid=30

输入参数

名称 类型 是否必须 示例值 默认值 描述
goodsid Integer 必须 1 商品的id

返回结果

名称 类型 是否必须 默认值 描述
result int 必须 1 如果等于“0”，删除失败，如果等于“1”，删除成功
message String 返回信息
