搜索CMS数据列表

标签ID

dataSearchTag

说明

可以通过向其传递参数来搜索，参数的名字为模型的字段英文名，传递方式可以通过form方式或url传参。

标签内输入参数

名称 类型 是否必须 示例值 默认值 描述
modelid int 是 1 无 cms模型id
catid int 否 无 同意模型ID下的不同caitd下的文章列表

返回结果

名称 类型 描述
数据列表的分页对象 Page 分页列表中的对象为Map，其中key为cms模型的字段英文名，
如通过.title可输出模型中字段名为title的字段值
