游客订单查询标签

标签ID

SearchOrderTag

标签内输入参数

名称 类型 是否必须 示例值 默认值 描述
ship_name String 是 张三 收货人姓名
ship_tel String 是 13111111111 必填 收货人电话
action 是 当提交form表单的时候，必须填写
name="action" value="search"

返回结果

名称 类型 描述
游客订单列表 Map<order> 此返回值是Map型，其关键字如下：
ordersPage 订单分页列表 ：ordersPage.result值为List类型
totalCount 订单总数
ship_name 收货人姓名
ship_tel 收货人电话
pageSize 分页大小
page 当前页
