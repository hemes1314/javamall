GoodsGallery 商品相册

需要简述

返回所有的商品相册列表参数

输入参数

名称 类型 描述
img_id int 图片id
goods_id int 商品id
thumbnail String 列表尺寸，用于各商品列表中
small String 小尺寸，用于商品详细中preview
big String 大尺寸
original String 原尺寸
tiny String 极小的，用于商品详细页中主图下的小列表图
isdefault int 是否是默认(0否1是)
