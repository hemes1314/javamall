会员订单列表标签

标签ID

MemberOrderListTag

说明

输出当前会员的订单列表

标签内输入参数

名称 类型 是否必须 示例值 默认值 描述
无

返回结果

名称 类型 描述
订单列表 Map 会员订单列表，此数据为Map型，其中的Key如下：
totalCount：Long型，表示订单条数
pageSize:int型，每页订单条数。
page：String型，共有多少页。
ordersList：List集合，订单列表。
status:Sting行，表示订单状态
keyword:Sring型，查找订单关键字。
