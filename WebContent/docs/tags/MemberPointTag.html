MemberPointTag 会员我的积分标签

  • 输入参数
  • 返回结果
  • 标签测试工具
输出当前会员 的积分情况，包括使用积分、冻结积分、积分明细等，此标签要在会员登录后才可以使用。

标签ID

MemberPointTag

标签内输入参数

名称 类型 是否必须 示例值 默认值 描述
无

返回结果

名称 类型 描述
action=null Map 此状态下输出会员的消费积分、等级积分、等级积分，目前等级、下一级等级名称
此返回值是Map型，其关键字如下：
freezepoint:消费冻结积分。
freezemp:等级冻结积分。
member:会员积分。
memberLv：会员等级。
nextLv：会员下一个等级。
action=list Map 此状态下输出的是会员积分明细
此返回值是Map型，其关键字如下：
pointHistoryList:List型，输出积分明细列表。
pageSize:int型，每页条数。
page：String型，共有多少页。
totalCount：Long型，表示条数
action=freeze Map 此状态下输出的是会员冻结明细
此返回值是Map型，其关键字如下：
pointFreezeList:List型，输出冻结明细列表。
pageSize:int型，每页条数。
page：String型，共有多少页。
totalCount：Long型，表示条数。
point：Integer型，会员积分
